Arcane Studio

Smart automations for development teams

Arcane Studio is a collection of productivity-boosting tools built on top of the Arcane Engine - it takes natural language instructions - Tasks - and executes them on your behalf using its capabilities, which include:

  • Reading, writing and editing code
  • Interacting with issues, tickets, wikis, etc via Integrations
  • Searching the internet and scraping websites

This studio uses tasks as a basic unit of work to create higher-level automations that can be shared and reused across teams. Everything you see is built using our Development Kit, which makes it easy for you to build your own tools.
